Immersive Course Structure

The .NET Core with Azure LOT provides exposure to the entire spectrum of .NET Core with Azure technologies, focusing on both desktop and cloud application development. Below is the 40-day immersive course structure:

  • Power Skills - 5 Days
  • Programming Foundation - 3 Days
  • .NET Core + C# 10.0 - 6 Days
  • Git - 1 Day
  • RDBMS and SQL Server - 3 Days
  • LINQ and Entity Framework Core - 2.5 Days
  • ASP.NET Core MVC - 4 Days
  • Azure DevOps - 1 Day
  • Cloud Computing & Azure Fundamentals - 2 Days
  • Cloud Networking and Storage - 2 Days
  • Azure PaaS Services - 3 Days
  • Sprint Implementation - 4 Days
  • Module Test + Sprint Evaluation - 1 Day
  • L1 Preparation - 1 Day
  • L1 Test - 1 Day
